Built an end-to-end LinkedIn content automation using n8n that converts raw AI news into ready-to-post content.

The system fetches latest AI articles via RSS, filters and selects the most relevant ones for SMBs using LLM, then scrapes full content and generates a structured LinkedIn post with summaries and links.

It also includes a human-in-the-loop approval system via email, allowing edits before final publishing, and automatically posts to LinkedIn once approved.

This workflow reduces manual research, content writing, and posting effort into a fully automated pipeline with control and scalability.
